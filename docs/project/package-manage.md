

### monorepo


单个库托管多个项目 

好处：
- 代码共享
- 一致性
- 提升协作效率


问题：
- 项目规模过大 


实际中解决的问题：
- 技术的积累沉淀，组件库 工具库 等 
- 项目间的依赖关系
