## 响应式 & 自适应

响应式: 
  - 通过灵活布局，依靠媒体查询等技术，让网页在所有屏幕尺寸上自动调整布局  
  - 提供一致的用户体验，所有设备都能得到优化的布局 
  - 
自适应: 
  - 基于特定断点设计多个固定布局，针对不同设备和屏幕尺寸加载特定的预定义布局 
  - 为特定尺寸设计的固定布局

实现: 
- 媒体查询 
- flex
- grid
结合相对单位(百分比， vw/vh)


具体:
1. tailwindCss + variable
