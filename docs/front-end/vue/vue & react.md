

## vue & react

### 相同点

- 组件化
- vdom
- 声明式UI 数据驱动视图
- 响应式，单向数据流
- 社区成熟，都有ssr

### 不同


vue 
- 渐进式框架 轻量 灵活



react
- 用于构建用户界面的js库
- 提供了一种声明式编程风格，更易于阅读和调试

*设计理念*
    - react FP 数据不可变
    - vue 灵活易用 渐进式框架，进行数据拦截，自动检测数据的变化并更新

*响应式实现不同*
    - react 
        - 基于状态机，手动优化，需手动setstate
    - vue
        - 数据劫持+订阅发布

*diff*
    -  都是深度遍历,
    - vue 变比对，变更新
    - react fiber架构，先patch 在批量更新

*写法*
    - css in js
    - sfc
