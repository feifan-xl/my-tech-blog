### 浪淘沙·最怕问初衷

最怕问初衷，幻梦成空 
莫问少年江湖梦，谁不少年梦江湖  
年少励志三千里，踌躇百步无寸功  
懵懂半生，庸碌尘世中 
转眼高堂皆白发，儿女蹒跚学堂中  
碎银几两催人老，心扔少
皱纹情然上眉中
浮生醉酒回梦里，青春仍依旧
只叹时光太匆匆
